### Booking Workflow

The current booking form sends booking requests to WhatsApp. Keep this functionality unchanged, but extend the workflow so that every booking submission:

1. Sends the booking details to WhatsApp.
2. Saves the booking in the application's database.
3. Makes the booking immediately available in the admin dashboard.

The booking data to capture includes:

* First Name
* Phone Number
* Email (optional)
* Service
* Preferred Date
* Preferred Time
* Notes
* Submission Timestamp
* Booking Status
